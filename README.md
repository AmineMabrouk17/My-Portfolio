# My Portfolio

Personal portfolio site built with Next.js.

**Live:** [my-portfolio-seven-chi-94.vercel.app](https://my-portfolio-seven-chi-94.vercel.app)

## Tech Stack

- Next.js 16 (App Router)
- TypeScript
- Tailwind CSS v4
- pnpm

## Getting Started

```bash
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000).

## Routes

- `/` — Main portfolio
- `/landing-1`, `/landing-2`, `/landing-3` — Arabic course landing pages
- `/concepts/gumroad` — Gumroad marketplace concept
- `/concepts/redeemly` — Redeemly topup platform concept

## Scripts

```bash
pnpm dev      # Start dev server
pnpm build    # Build for production
pnpm start    # Start production server
pnpm lint     # Run ESLint
```
